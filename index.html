01

HTML & CSS

Základy webu — štruktúra, štýly, layout, responzivita

Stage 1 – 2
HTML základy
Elementy, atribúty, linky, obrázky, zoznamy, tabuľky, formuláre
HTML

Čo prejsť

W3Schools HTML Tutorial — sekcie Elements, Attributes, Headings, Paragraphs, Links, Images, Lists, Tables, Forms. Netreba čítať všetko — prejdi príklady, skúšaj v editore.

→ Úloha

Vytvor stránku „O mne" — nadpis, odsek o sebe, zoznam 5 obľúbených vecí, obrázok, odkaz na obľúbenú stránku. Žiadne CSS, len čisté HTML. Má to byť škaredé — to je v poriadku.

Stage 3 – 4
CSS základy
Selektory, farby, pozadia, box model, fonty, štýlovanie textu
CSS

Čo prejsť

W3Schools CSS Tutorial — Selectors, Colors, Backgrounds, Borders, Margins, Padding (box model!), Fonts, Text, Links. Kľúčové je pochopiť box model — margin, border, padding, content.

→ Úloha

Vezmi svoju „O mne" stránku a uprav ju CSSkom. Zmeň farby, fonty, pridaj pozadie. Cieľ: aby vyzerala ako niečo, čo by si ukázal kamarátovi.

Stage 5
Flexbox
Flexbox layout + hra Flexbox Froggy
CSS

Čo prejsť

W3Schools CSS Flexbox a potom Flexbox Froggy — hra, kde CSSkom naviguješ žaby na lekná. Celá hra trvá ~30 minút a naučí flexbox lepšie ako hodina čítania.

→ Úloha

Preusporiadaj svoju stránku pomocou flexboxu — daj obsah do stĺpcov, vycentruj veci, sprav navigačnú lištu hore.

Stage 6
CSS Grid + Responsive
Grid layout, media queries, mobilná verzia
CSS

Čo prejsť

W3Schools CSS Grid (len basics), Grid Garden (hra na grid), a W3Schools Media Queries.

→ Úloha

Pridaj na stránku grid galériu (3×2 mriežka obrázkov/kariet). Stránka sa musí rozumne zobraziť aj na mobile — otestuj cez DevTools (F12 → toggle device toolbar).

Stage 7
Projekt — Osobná stránka v1.0
Postav od nuly vlastnú personal website
HTMLCSS

Požiadavky

Navigácia hore · Hero sekcia (veľký nadpis + popis) · Sekcia „O mne" · Sekcia „Projekty" (placeholder) · Footer s kontaktom.

→ Pravidlo

Žiadne copy-paste z tutoriálov. Písať od prázdneho súboru. Keď nevieš niečo — Googluj, ale nekopíruj celé bloky. Ak to ide ťažko, je to normálne — práve sa učíš.

02

JavaScript + Finálny projekt

Interaktivita, DOM manipulácia, a vylepšenie tvojej stránky

Stage 8 – 9
JavaScript základy
Premenné, funkcie, eventy, DOM manipulation
JS

Čo prejsť

W3Schools JavaScript — Variables (let, const), Data Types, Functions, Events, DOM (getElementById, querySelector, innerHTML, addEventListener). S Pythonom to pôjde rýchlo — syntax je iná, ale logika je rovnaká.

→ Úloha

Sprav stránku s tlačidlom „Zmeň farbu pozadia" — každý klik nastaví náhodnú farbu. Bonus: pridaj counter, koľkokrát si klikol.

Stage 10
DOM + Eventy
Vytváranie elementov, klikanie, input handling
JS

Čo prejsť

W3Schools JS HTML DOM a JS Events. Kľúčové: createElement, appendChild, remove, classList.toggle.

→ Úloha

Sprav to-do list: input pole, tlačidlo „Pridaj", zoznam pod tým. Klik pridá položku. Bonus: klik na položku ju prečiarkne.

Stage 11
Interaktívne prvky
Časovače, localStorage, mini-projekt podľa výberu
JSCSS

Čo prejsť

W3Schools: setTimeout, setInterval, a localStorage. Tieto tri veci odomknú množstvo možností.

Vyber si 1 mini-projekt

Stopky — štart, stop, reset, reálny čas
Quiz — 5 otázok, skóre na konci
Dark/Light mode — prepína tému, pamätá si voľbu

Stage 12 – 14
Finálny projekt — Stránka v2.0
Vylepši svoju osobnú stránku o interaktívne prvky
HTMLCSSJS

Pridaj minimálne 3 z týchto

✦ Smooth scroll navigácia
✦ Dark/light mode toggle
✦ Animovaný hero — text sa postupne objaví
✦ Interaktívna sekcia (quiz, kalkulačka, generátor faktov)
✦ Galéria s lightboxom (klik → zväčší obrázok)
✦ Kontaktný formulár s validáciou (JS skontroluje email)

→ Na konci

Odprezentuj stránku — čo si spravil, čo bolo najťažšie, čo by si spravil inak. Toto je tvoj prvý web.
